📖 LIMIT VWAP 자동 매매 사용법

본 시스템은 실시간 가격 가중 평균선인 VWAP(Volume Weighted Average Price)을 기준으로 가격 이탈 시 지정가 진입/청산을 자동 수행하는 봇입니다.

📌 탭 구분 설명

  • 📊 종합 자산 현황: 가상 봇의 메모리 상 모의 자산과, 실제 API를 연동한 실계좌 평가 현황 및 보유 포지션을 병렬 요약 표출합니다.
  • 🤖 가상 자동매매: 안전 샌드박스 가상 환경에서 가상 시작원금으로 모의 체결 및 Cancel & Replace 메커니즘을 테스트합니다.
  • 💼 실제 자동매매: 토스증권 Open API를 활용한 실제 주문 체결 환경입니다. API 자격증명이 저장되어야 활성화됩니다.

⚠️ 주의 사항 및 안전 장치

  • 당일 손실 한도: 당일 시작 자산 기준 지정된 손실 한도(%) 이탈 시, 봇은 모든 주문을 강제 취소하고 시장가 전량 매도 청산(Panic Exit) 후 자동 정지됩니다.
  • 투자 비중(K%): 1회 매수 시 지정한 비율만큼만 잔고를 할당하여 신용 융자나 미수 거래 발생 위험을 원천 차단합니다.
