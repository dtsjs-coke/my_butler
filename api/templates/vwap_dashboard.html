📖 LIMIT VWAP 자동 매매 매뉴얼

본 자동 매매 프로그램은 실시간 VWAP (Volume Weighted Average Price, 거래량 가중 평균 가격)을 기준선 삼아 지정가 매매를 수행하고 정정(Cancel & Replace)을 자동화하는 고성능 봇입니다.

📌 핵심 매매 로직 흐름

  • 1. 진입 (매수 지정가 배정): 현재가(종가)가 실시간 VWAP 값보다 아래에 위치해 있을 때, 봇은 이격 목표가인 VWAP - N% 가격으로 지정가 매수 주문을 냅니다. 만약 VWAP 값이 변경되면 기존 미체결 주문을 취소하고 신규 목표가로 정정 주문(Cancel & Replace)을 반복 집행합니다.
  • 2. 청산 (매도 지정가 배정): 매수 체결 후 포지션을 잡으면, 봇은 즉시 VWAP + M% 가격(혹은 VWAP 선 돌파 가격)에 매도 지정가 주문을 배치하여 익절 대기를 시작합니다. 이 또한 실시간 가격 변동에 맞춰 자동으로 계속 추종 정정됩니다.
  • 3. 시장가 손절 (Stop Loss): 급격한 폭락 시 손실 한도를 가두기 위해, 주식 매수 평단가 대비 -X% 이하로 주가가 하락하면 기존 미체결 주문을 즉각 취소하고 시장가 전량 청산을 단행합니다.

💡 퀵 스타트 가이드

  • 모의 테스트 실행: 우측 상단의 `⚙️ API & 보안 설정` 버튼을 눌러 본인의 암호를 변경한 뒤, 거래 모드를 `VIRTUAL`로 두고 이격률을 작게 잡아(예: N=0.1%, M=0.5%) 봇이 지정가를 어떻게 계속 갱신하는지 로그를 통해 지켜봅니다.
  • 실거래 투입: 검증이 완료되면 `REAL` 모드로 변경하고, `⚙️ API & 보안 설정` 모달에 토스증권 API Key 및 계좌번호 식별 코드를 입력해 실시간 주문 체결 상태로 전환합니다.
