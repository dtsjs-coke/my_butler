# LIMIT VWAP Trading Bot 사용자 가이드 (User Manual)

본 문서는 LIMIT VWAP 자동 매매 시스템을 처음 접하는 사용자를 위한 상세 운영 매뉴얼입니다.

---

## 1. LIMIT VWAP 전략 핵심 요약

본 자동 매매 봇은 당일 장 시작 시각부터 실시간으로 **VWAP(거래량 가중 평균 가격)**을 계산하고, 이 VWAP 값을 기준선 삼아 철저히 **지정가(Limit)** 기반의 매매 및 미체결 정정(Cancel & Replace)을 수행합니다.

### 📈 매매 규칙 개요
1. **진입 (지정가 매수)**:
   * 현재가가 VWAP 선보다 아래에 위치해 있고 포지션이 없을 때 작동합니다.
   * 봇은 실시간으로 **매수 타겟 가격**($TargetBuy = VWAP \times (1 - N\%)$)을 계산합니다.
   * 이 타겟 가격에 **지정가 매수** 주문을 걸어놓습니다. VWAP 선이 변동함에 따라 미체결 매수 주문은 계속 취소 및 재접수(Cancel & Replace)를 반복하며 따라갑니다.
2. **청산 (지정가 매도)**:
   * 주식을 보유하고 있을 때 작동합니다.
   * 현재가가 VWAP 선을 다시 상향 돌파하거나, **매도 타겟 가격**($TargetSell = VWAP \times (1 + M\%)$) 이상으로 상승할 경우 **지정가 매도** 주문이 체결되거나 시장가로 청산됩니다.
3. **손절 (Stop Loss)**:
   * 주식 매수 평단가 대비 가격이 $-X\%$ 이하로 급락하면, 즉시 미체결 매도 주문을 취소하고 **시장가(Market)로 전량 청산**하여 추가 손실을 방지합니다.

## 2. 대시보드 탭 메뉴 설명

대시보드 상단 탭을 통해 세 가지 주요 영역을 쉽게 제어할 수 있습니다.

### ① 📊 실시간 대시보드
* **실시간 모니터링**: 현재가, VWAP, 계산된 매수/매도 타겟가와 함께 현재 전략 시그널(`BUY`, `SELL`, `HOLD`, `WAIT`) 상태가 2초 주기로 갱신됩니다.
* **봇 제어 버튼**: `봇 가동 (START)` 및 `봇 중지 (STOP)` 버튼으로 백그라운드 트레이딩 엔진을 켜고 끌 수 있습니다.
* **미체결 주문**: 현재 시장에 걸려있는 지정가 주문 현황을 실시간으로 확인합니다.
* **자산 및 성과**: 총자산 평가금액, 투자 원금 대비 **누적 수익률(ROI %)**, 보유 종목의 평가 손익을 실시간 변동률과 함께 한눈에 볼 수 있습니다.
* **체결 히스토리 & 로그**: 거래가 체결될 때마다 이력이 아래에 누적되며, 우측의 터미널을 통해 봇이 뒤에서 수행하는 모든 연산 및 정정 작업 로그(`trading_bot.log`)가 실시간으로 자동 스크롤 출력됩니다.

### ② 📈 과거 백테스트
* 토스증권 API의 최근 과거 차트(150봉) 데이터를 내려받아, 해당 종목에 사용자가 설정한 $N, M, X$ 파라미터를 대입했을 때의 역사적 수익률, MDD(최대 낙폭), 승률, 거래 횟수를 시뮬레이션하여 5초 만에 성과 보고서를 뽑아냅니다.

### ③ ⚙️ API 및 보안 설정 (모달)
* 대시보드 우측 상단의 `⚙️ API & 보안 설정` 버튼을 누르면 나타납니다.
* 토스증권 Open API 연동에 필요한 `Client ID`, `Client Secret`, `계좌식별자(Account Seq)` 정보와 대시보드 로그인 패스워드를 변경할 수 있으며, 디스크 저장 시 **대칭키 암호화**되어 철저히 은닉됩니다.

## 3. 매매 파라미터 상세 정의

* **거래 모드**: 
  * `VIRTUAL` (가상): 실제 자산에는 전혀 영향이 없으며, 설정된 '가상 원금'을 활용해 메모리 상에서 실시간 시세를 추종해 매수/매도 체결을 모사하는 안전 Sandbox 모드입니다.
  * `REAL` (실제): 실제 토스 계좌의 자금으로 실제 지정가 주문이 집행되므로 신중하게 구동해 주십시오.
* **시장 구분**: `US` (미국 주식) 또는 `KR` (한국 주식)을 선택합니다.
* **종목 코드**: 미국 주식은 티커(ex: AAPL, TSLA), 한국 주식은 6자리 종목번호(ex: 005930)를 입력합니다.
* **VWAP 리셋 시각**: 매일 누적 계산이 리셋되어 새롭게 계산이 시작되는 시각을 의미합니다. (미국 주식 본장 기준 `22:30` 또는 `23:30`, 한국 주식 기준 `09:00` 입력 권장)
* **봉 주기**: 데이터 수집 및 계산에 활용할 봉의 크기입니다. (1분봉, 5분봉 등)
* **투자 원금**: 가상 및 백테스트 실행 시 기준이 되는 가상의 시작 자본금입니다.
* **매수 이격률 (N%)**: 현재가가 VWAP 대비 하방으로 몇 % 이상 벌어졌을 때 진입할 것인지 설정합니다.
* **매도 이격률 (M%)**: 주식을 보유한 상태에서 현재가가 VWAP 대비 상방으로 몇 % 이상 올랐을 때 청산할 것인지 설정합니다.
* **손절 비율 (X%)**: 주식 매수 평단가 대비 하락 시 즉시 강제 청산할 손절 한도%입니다.
* **투자 비중 (K%)**: 보유 현금 예수금 중 한 거래당 최대 얼마만큼의 비율(%)을 매수에 활용할 것인지 제어하여 분할 매매 자금 관리를 돕습니다.
* **최대 가용 한도액 (Limit)**: 계좌 잔고가 아무리 많더라도 1회 주문당 이 금액 이상으로는 절대 매수 주문을 제출하지 않도록 한계선을 강제합니다.

## 4. 첫 구동을 위한 퀵 스타트 가이드

1. **로그인**: 웹 브라우저로 `/vwap` 접속 후 관리자 암호(초기값: `admin1234`)를 입력해 로그인합니다.
2. **보안 설정**: 우측 상단 `⚙️ API & 보안 설정` 버튼을 누르고, 본인의 비밀번호를 변경하거나 실제 거래를 돌릴 경우 토스 API 키를 입력하고 저장합니다.
3. **가상 테스트 세팅**: 
   * 거래 모드: `VIRTUAL`
   * 종목 코드: `AAPL` (또는 원하는 미국/한국 주식)
   * 매수 이격률 `N`: `0.1` (테스트를 위해 매우 좁게 설정하여 즉시 주문 발동을 확인하기 위함)
   * 매도 이격률 `M`: `0.5`
   * 손절 비율 `X`: `2.0`
   * 투자 비중 `K`: `10.0` (가상 잔고의 10%만 진입)
4. **저장 및 가동**: `설정 및 API 키 동적 저장`을 누르고, 상단에서 `봇 가동 (START)`를 활성화합니다.
5. **모니터링**: 약 1~5분 이내에 봇이 가격의 변동에 맞춰 미체결 매수 지정가를 계속 정정(Cancel & Replace)하는 모습과 로그가 터미널에 흘러가는지 모니터링합니다.

## 5. 금융 안전장치 및 미수/신용 거래 방지 정책

본 자동매매 시스템에는 의도하지 않은 자금 손실 및 빚(레버리지) 거래를 방지하기 위해 이중화된 **금융 안전장치(Safety Guard rails)**가 봇 코어에 기본적으로 탑재되어 있습니다.

* **최대 투자 한도 상한선 강제**: `최대 가용 한도액 (Limit)` 파라미터를 통해 1회 주문당 투입될 자금의 절대적인 상한선을 둡니다. 잔고 평가액이 수억 원이라 하더라도 사용자가 설정한 한도(예: 100만 원)를 초과해 주문하지 않습니다.
* **순수 현금 잔고(Cash) 기반 예산 검증**: 미수 거래 및 신용 융자를 차단하기 위해, 봇은 매수 주문 직전 계좌의 **순수 보유 현금(예수금)** 상태를 이중 점검합니다. 주문 예정 총액($TargetBuyPrice \times Quantity$)이 실제 현금 잔고를 초과할 경우 매수 주문은 즉시 차단되거나 가용 잔고 범위로 강제 축소됩니다.
* **현금 100% 매수 주문 보장**: 토스증권 API 호출 시 매수 주문 형태를 신용융자가 불가능한 **현금 주문(CASH)**으로 명시하여, 빚을 내는 방식의 레버리지 거래 발생 위험을 원천 무력화합니다.
